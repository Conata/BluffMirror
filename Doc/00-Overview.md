# FPS Trump Game - ゲーム概要

## プロジェクト名
**FPS Trump Game** - 心理戦ババ抜き

## コアコンセプト

### 目指す体験
- **没入感**: 「カードUI」ではなく「テーブルの向こうに相手がいる」リアルな対戦体験
- **心理圧**: AIのセリフが空間に浮き、プレイヤーの判断に心理的圧力をかける
- **演出のみの影響**: セリフは「止める」「煽る」「誤誘導する」が、ゲームロジックには影響しない

### ゲームの特徴
1. **FPS視点**: プレイヤーは座っている目線でテーブルを見下ろす
2. **物理的操作**: カードを直接つまんで引く感覚
3. **心理戦演出**: AIが空間に投影するセリフで心理戦を演出
4. **ミニマルUI**: 従来のゲームUIを排除、没入感を最優先

## ターゲット体験

### プレイヤーが感じるべきこと
- 「本当にテーブルに座っている」没入感
- AIの視線や仕草による緊張感
- カードを「掴む」「引く」物理的な手触り
- 心理的駆け引きによる高揚感

### 技術的目標
- 60fps安定動作
- 遅延感のない操作レスポンス
- リアルタイムライティング
- 空間音響による臨場感

## 競合との差別化

### 従来のカードゲーム
- UI重視、画面レイアウト中心
- ゲーム性重視、演出は二の次

### 本ゲーム
- 体験重視、没入感最優先
- 演出でゲーム性を引き上げる
- FPS的操作感でアクション要素追加

## 成功指標
- プレイヤーが「もう一回」と自然に思う
- AIとの心理戦に夢中になる
- 従来のカードゲームUIに戻れなくなる
